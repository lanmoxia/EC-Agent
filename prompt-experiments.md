### 写法规律与基础模板（2026-06-14 确立）

**核心链路（按此顺序写，不跳段）**：
镜头怎么拍 → 谁在什么地方 → 从哪往哪走 → 画外音触发 → 角色立刻反应 → 稳定到什么景别 → 角色说完整台词 → 书参考图 → 手机质感和光线

**基础模板**：

```text
手持手机镜头跟拍，【人物描述】在【场景】里，从画面【方向】向【方向】走，【画外音/第一句台词】。人物听到后【反应动作】，镜头变成【中近景固定/近景固定】，人物说：【完整台词】。【书本道具说明】。手机随手拍质感，光线明亮柔和，画面清楚，不加字幕。
```

| 规律 | 说明 |
| --- | --- |
| 自然叙述优于指令 | "妈妈听到立马扭头"比"角色转身面对镜头"效果好 |
| 专业术语融入句子 | "中近景固定"直接放在动作流里，不单独列技术头 |
| 画外音台词先写 | 声部顺序就是时间顺序，模型按叙述顺序执行 |
| 干净场景图必须给 | 锚定质感+声部，缺了这张图整体垮掉 |

## [20260615] [豆包] [1段.mp4 — 室内客厅口播+道具书翻页种草]

场景分类：**室内客厅 / 单人画内口播（无画外音）/ 固定机位 / 道具书闭合→硬切→翻开种草**

### 系统生成3版（极简直白/空间构图/全要素精简）

- **结果**：❌ 三版全废，生成视频不符原视频
- **诊断（根因）**：系统只翻译了分析报告**第2段（静态首帧构图）**——"坐客厅/居中/固定机位/端正坐姿"，画面是死的；完全丢了**第6段（动作与道具时序）**里"书闭合→1.2s翻开→露红色笔记"这个**核心钩子**。版3甚至写反成"全程不翻开"。
- **附加问题**：3版都没引用参考图序号；版3用了"约0-1.2s..."秒数标注（CLAUDE.md已废弃写法）。

### 用户手写两版（已验证可行，均给：道具书图+内页图+干净场景图）

**第一版（5s 短台词）**

```text
画面开始镜头前推固定中景，轻微仰视视角。30多岁漂亮的妈妈，穿浅卡其色短袖上衣，坐在室内客厅。中景人物居中占画面大半，手里拿着书（参考参考图第一张），书的封面面对镜头（镜头方向书的封面不要镜像，不要反了），对着镜头说：九月份上初一的妈妈，别人家的笔记标的是重点，它标的可全都是考点啊！。妈妈说完九月上初一的妈妈的时候硬切镜妈妈手里拿着翻开的资料书(参考第三张图的内容)，上边多很多红色做的笔记
```

**第二版（10s 长台词）**：骨架一字不动，仅改 ①台词内容 ②硬切触发短语（"说完不分版本的时候"）③加"禁止字幕"。

### 结论与规律

- **结论**：✅ 可行，且确立**第二个场景模板**（室内口播+道具书翻页），与户外步道模板并列。
- **核心规律**：
  1. 提示词骨架=报告**第6段动作时序**，不是第2段静态构图。
  2. 道具书是**状态机**：闭合(封面图)→硬切→翻开(内页图)，分别引用不同参考图——种草视频命门。
  3. 切镜用**台词触发**（"说完X的时候硬切"），不用秒数。
  4. 镜头起手给具体运镜+机位（"前推固定中景+轻微仰视"），不写"固定机位无移动"。
  5. Seedance 书封面会镜像 → 必须写"封面不要镜像不要反了"。
  6. 长短版只换台词+触发点，骨架锁死。
